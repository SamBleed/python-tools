# Python Tools - Colección de Scripts Útiles

Este repositorio contiene una colección de scripts en Python diseñados para facilitar tareas comunes relacionadas con redes, sistema y seguridad. Son herramientas pensadas para automatizar y agilizar actividades frecuentes en entornos técnicos y de pentesting.

---

## Contenido

- `01-info_sistema.py` — Obtiene información básica del sistema y red.
- `02-mac_changer.py` — Cambia la dirección MAC del adaptador de red.
- `03-wifi_reset.py` — Reinicia el adaptador WiFi.
- `04-nmap_launcher.py` — Lanza un escaneo básico con Nmap.

---

## Requisitos

- Python 3.x
- Módulo `subprocess` (viene por defecto con Python)

---

## Uso

Para ejecutar cualquiera de los scripts, abre una terminal y ejecuta:

```bash
python nombre_del_script.py
````

Por ejemplo, para cambiar la MAC:

```bash
python 02-mac_changer.py
```

---

## Consideraciones

* Ejecutar con permisos de administrador para que algunas funciones (como cambiar la MAC o reiniciar adaptadores) funcionen correctamente.
* Estos scripts son para fines educativos y pruebas en entornos controlados.
