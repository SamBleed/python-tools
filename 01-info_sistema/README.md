# 01-info_sistema.py

Este script en Python obtiene información básica del sistema y la red, incluyendo detalles como:

- Nombre del equipo
- Dirección IP local
- Dirección MAC del adaptador principal
- Sistema operativo

---

## Requisitos

- Python 3.x

---

## Uso

Ejecuta el script desde la terminal con:

```bash
python 01-info_sistema.py
````

---

## Notas

* Este script está pensado para uso educativo y en entornos controlados.
* Puede requerir permisos de administrador para acceder a ciertos datos en algunos sistemas.
