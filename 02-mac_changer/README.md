# 02-mac_changer.py

Este script en Python permite cambiar la dirección MAC del adaptador de red principal de tu computadora. Es útil para pruebas de seguridad y privacidad en redes.

---

## Requisitos

- Python 3.x  
- Ejecutar con permisos de administrador (sudo en Linux, CMD/PowerShell como administrador en Windows).

---

## Uso

Ejecuta el script desde la terminal con:

```bash
python 02-mac_changer.py
````

El script te pedirá la nueva dirección MAC que deseas asignar.

---

## Advertencias

* Cambiar la MAC puede desconectar temporalmente tu conexión de red.
* Usa este script únicamente en redes y dispositivos donde tengas permiso para realizar estas acciones.
* Está destinado para fines educativos y pruebas en entornos controlados.
